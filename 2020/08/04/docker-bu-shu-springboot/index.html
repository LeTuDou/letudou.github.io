Docker部署SpringBoot

Docker配置

# Centos7开启方式
vim /lib/systemd/system/docker.service
# 找到 ExecStart并修改
ExecStart=/usr/bin/dockerd -H tcp://0.0.0.0:2375 -H unix://var/run/docker.sock
# 重启docker
sudo systemctl daemon-reload #守护进程重启
systemctl restart docker #重启docker服务
# 检验
curl http://localhost:2375/version

编辑Dockerfile

项目根目录创建docker文件夹

编辑Dockerfile文件

FROM java:8u111
MAINTAINER xiaozhang letudou@qq.com
VOLUME /temp
ADD *.jar app.jar
RUN bash -c 'touch /app.jar'
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "app.jar"]

  1. 另外项目端口需要改为8080

  2. 对Dockerfile里面的内容做一个简单的介绍:

     1.FROM:指明当前镜像继承的基镜像,编译当前镜像时候会自动下载基镜像
 2.MAINTAINER:当前镜像的作者和邮箱,使用空格隔开
 3.VOLUME:挂载目录
 4.ADD:从当前工作目录复制文件到镜像目录中并重新命名
 5.RUN:在当前镜像上执行Linux命令,这里我执行了2个run指令
 6.EXPOSE:监听的端口号
 7.ENTRYPOINT:让容器像一个可执行程序一样运行
